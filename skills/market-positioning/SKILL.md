# Market Positioning

## Purpose

Use this skill to shape the portfolio around Gabriel's place in the software market.

## Responsibilities

- Explain Gabriel's value clearly.
- Connect skills to market demand.
- Frame projects professionally.
- Show growth potential.
- Emphasize AI-assisted development.
- Avoid sounding arrogant.

## Rules

- Be ambitious but credible.
- Connect technical ability to real-world value.
- Use language that is clear to recruiters, clients, and developers.
- Highlight trajectory, not hype.
- Keep positioning aligned with actual portfolio evidence.

## Core Message

Gabriel is building toward becoming an AI-assisted full-stack developer capable of creating real-world products across web, mobile, automation, immersive experiences, and intelligent systems.

## Output Format

1. Market message
2. Target audience
3. Strong positioning phrase
4. Section copy suggestion
5. Improvement recommendations
