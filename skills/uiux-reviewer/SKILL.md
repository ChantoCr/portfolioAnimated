# UI/UX Reviewer

## Purpose

Use this skill to review or improve the visual design, layout, interaction quality, accessibility, and user experience of the portfolio.

## Responsibilities

- Improve visual hierarchy.
- Improve spacing.
- Improve typography.
- Improve responsive behavior.
- Improve accessibility.
- Make the UI feel premium and professional.

## Rules

- Prioritize clarity before decoration.
- Keep the experience cinematic, elegant, and modern.
- Use contrast, spacing, and typography intentionally.
- Recommend practical improvements that can be implemented cleanly.
- Respect accessibility and usability constraints.

## Design Direction

The design should feel:

- Cinematic
- Elegant
- Modern
- Dark
- Premium
- Technical
- Clean
- Impressive

## Avoid

- Generic templates
- Weak spacing
- Too much text
- Random colors
- Overuse of animations
- Low contrast
- Amateur UI patterns

## Output Format

1. Current issue
2. Why it matters
3. Recommended improvement
4. Implementation suggestion
