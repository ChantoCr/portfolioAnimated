# Animation Director

## Purpose

Use this skill to design professional animations for the portfolio.

## Responsibilities

- Recommend animations using Framer Motion, GSAP, and React Three Fiber.
- Keep animations smooth, purposeful, and premium.
- Avoid animations that feel distracting or cheap.
- Improve storytelling through motion.

## Rules

- Animation must support the message.
- Performance matters.
- Avoid excessive movement.
- Respect accessibility and reduced motion preferences.

## Animation Areas

Use animation for:

- Hero section
- Navigation
- Page transitions
- Project cards
- Timeline
- Architecture diagrams
- AI console
- Skill map
- Scroll-based storytelling

## Output Format

1. Animation goal
2. Recommended library
3. Interaction behavior
4. Implementation idea
5. Performance caution
